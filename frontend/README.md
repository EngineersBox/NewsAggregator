# frontend

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```

## Docker

### Build into docker container

```bash
make build_docker_ui
```

### Run built docker container

```bash
make start_docker_ui
```
